Design and build a clean, modern, mobile-first UI for a worker finder app named "KaamMitra".

Important:

* Focus ONLY on the mobile app frontend UI and screen design.
* Do NOT build backend logic yet.
* Do NOT build payment system.
* Do NOT build admin panel yet.
* Design the app in a way that it can later connect to Firebase and an Admin Panel.
* Keep the UI professional, lightweight, beginner-friendly, and production-ready.
* Use modern cards, rounded corners, proper spacing, clean typography, and intuitive navigation.
* The design should feel similar to a real-world marketplace app.

App Purpose:
KaamMitra helps people find local workers and skilled professionals such as:

* Plumber
* Electrician
* Carpenter
* Mason
* Painter
* AC Technician
* Welder
* Labour
* CCTV Installer
* RO Technician
* Home Cleaning Worker
* Other local service providers

Users can browse workers, view profiles, and directly contact them through phone call or WhatsApp.

Design Style:

* Mobile-first design
* Clean and modern
* Professional Indian service marketplace feel
* Minimal color palette
* Soft shadows and rounded cards
* Fast and easy navigation
* No cluttered UI
* Production-ready appearance
